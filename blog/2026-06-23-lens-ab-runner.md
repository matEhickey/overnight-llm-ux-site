Lens A/B Runner takes the same prompt and runs it through two Perspective Deck lenses side by side.
Live metrics — chars, words, sentences, bullets, code blocks, named-entity density, and a heuristic verdict — make the difference visible instead of guessed.
A new NPC lens pack (Grumpy Wizard, Friendly Shopkeeper, Cyberpunk Hacker, Wise Mentor) gives the comparison some flavor to lean into.
