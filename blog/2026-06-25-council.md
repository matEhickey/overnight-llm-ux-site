The Council is a new mode that puts 2–4 NPC personas in the same room.
Pick the cast (defaults to all four — Grumpy Wizard, Friendly Shopkeeper, Cyberpunk Hacker, Wise Mentor), type one prompt, and each member responds in sequence through the same local model.
The result is a panel-discussion transcript where every voice is preserved verbatim, followed by a Consensus / divergence card that surfaces the shape of the disagreement and a pairwise metric table.
The metric harness from the A/B runners is reused unchanged — only the runner changes.
