# Overnight Game

A 3D first-person exploration prototype built with React Three Fiber and Three.js — rebuilt from `main` each overnight cycle.

## How it works

Each night, a new feature is developed on a fresh branch starting from `main`:

1. **Analyse** — what exists, what's the state
2. **Research** — what would be cool and achievable
3. **Decide** — pick one feature
4. **Implement** — code the feature
5. **Build** — `npm run build` → static HTML
6. **Document** — blog post + docs article with live demo

Features are not merged automatically. Mathias reviews, tests from the blog, and merges manually.
